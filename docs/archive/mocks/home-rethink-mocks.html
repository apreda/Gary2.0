Home Rethink — assessment + two directions

Mocks to react to, nothing built. Both keep the things you've called perfect (results-first mornings, the live sweat leading during games, the recap voice) and differ in what the page fundamentally is.

My read on the current Home

  • The trust signal is buried. For a picks product, the record IS the brand — but the scorecard sits below the prop box, and Gary's rolling record (the 55% L7 the Hub masthead now carries) appears nowhere on Home. A new user's first question — "is he actually good?" — goes unanswered above the fold.
  • No new-user path. Day-0 users land on last night's recap carousel with zero framing: who Gary is, what's free (the daily free pick is the hook), where to start. The intro sheet only fires from other tabs.
  • Home duplicates the other tabs instead of routing to them. Today's Board ≈ Picks, the prop box ≈ props surfaces, Big Games ≈ Tomorrow page. Six ~equal modules stack with no hierarchy — it reads as "everything, again" rather than a front door with a point of view.
  • The day-arc is implicit. The code re-orders blocks by morning/pregame/live, which is the right instinct — but the page never says what moment it is. The fan's day is a story: last night → today's slate → the sweat → back to results. The page should tell it.
  • What's genuinely working: the live-sweat takeover, the recap headlines voice, the countdown hero, results-first mornings. All four survive in both directions below.
DIRECTION A · RECOMMENDED

“The Ledger” — one identity block, one moment, clear doors

Home becomes three fixed acts. Act 1 is a permanent identity/trust header — Gary's record, always, for everyone. Act 2 is ONE time-aware hero (morning story / free-pick + countdown / live sweat) instead of a stack of six modules. Act 3 is the day's accountability feed — every pick Gary posted today with a status chip (pending → live → ✓/✗) — plus three doors to the tabs that own the depth. Everything else Home currently renders gets cut or demoted to its owning tab.

DIRECTION B

“SportsCenter Open” — story-led rundown

Keeps today's DNA (the headlines carousel stays the lead) but reframes the page as a broadcast rundown: scores ticker on top, TOP STORIES bigger and story-first, then GARY'S DAY (same accountability feed as A), record as the sign-off. On day 0 the first story card is "Meet Gary" — the intro + record — so new users get framing inside the format instead of a separate sheet.

Why you might prefer it: it's the smaller change — the carousel stays the lead, and the new-user problem is solved inside the format (a Start Here story) rather than by restructuring the page. What I'd cut in either direction (and where it goes)

  • Today's Board (all-games table) → the Hub's slate strip + game sheets now do this job; Picks owns pick-by-game.
  • Prop Box → Winners/props surfaces own it; Home links to it from Gary's Day.
  • Big Games To Watch → merges into the hero (morning story / countdown) instead of a third standalone module.
  • Keep untouched: live-sweat takeover logic, DailyRecapOverlay, the recap voice, TOMORROW's look-ahead page.
