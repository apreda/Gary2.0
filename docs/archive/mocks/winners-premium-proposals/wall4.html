WRAPPERS ROUND 4 — objects, not stickers

Every wrapper is now a dimensional object: perspective, fold creases, ribbons with cast shadows, lids with thickness, cloth with real folds. HOVER to tilt · CLICK to unwrap. All buildable: these exact motions port to SwiftUI (rotation3DEffect + shadows) or — for the richest versions — a Rive animation file that plays identically in-app. Note on tools at the bottom.
